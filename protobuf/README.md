## ProtoBuf

#### PB介绍
```
Protobuf是Google公司内部的混合语言数据标准，他们用于RPC系统和持续数据存储系统。

Protobuf是一种轻便高效的结构化数据存储格式(类似json)，可以用于结构化数据串行化，或者说序列化,方便文件的存储与网络传输.
它很适合做数据存储或RPC数据交换格式。可用于通讯协议、数据存储等领域的语言无关、平台无关、可扩展的序列化结构数据格式。
目前提供了 go、Java、Python 等语言的 API。

类似json, protobuf也用来实现跨语言间的数据存储与传递.
```

#### 怎么使用PB
```
1.写一个proto的文件 .定义你需要的数据结构.

2.使用protoc编译器把写的proto文件编译为目标语言的相关类或结构体.
(目前google提供了 go、Java、Python等语言的 API).

3.把第二步生成的类包含到你写的程序中,就可以使用它了,存储传输数据.
```
